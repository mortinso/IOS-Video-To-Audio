# IOS-Video-To-Audio
IOS shortcut that receives a video from "Share Sheet" and saves a copy of the audio track to a `.m4a` file. 
On first open, the user will be prompted to pick the directory where the files should always be saved (default being "On My iPhone").
The user can trim which part of the original video they want the audio from.
The user can also choose a name for the file to get saved under.
